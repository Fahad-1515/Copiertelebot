# 📋 Copiertelebot - Telegram Forward Bot
A powerful Telegram bot that automatically forwards messages from one channel/group to another. Perfect for content aggregation, deal sharing, and team collaboration.

# 🤖 What This Bot Does
This bot copies messages from any Telegram channel/group and forwards them to your chosen destinations automatically. Think of it as a content aggregator that works 24/7.

# 🚀 Quick Start Guide

Step 1: Start the Bot

1. Open Telegram

2. Search for @Copiertelebot

3. Send /start command

Step 2: Create Your First Forwarding Job

1. Click "▶️ Start Forwarding"

2. Enter Source Channel - Where to copy messages from

  text
Examples:
• https://t.me/channelname
• @channelname
• https://t.me/channelname/200 (starts from message #200)

5. Add Destination(s) - Where to send messages

   text
Examples:
• @mychannel
• https://t.me/mychannel
• -1009876543210

6. Choose Message Count - How many messages to forward

 text

• 50 → forwards last 50 messages

• 200 → forwards last 200 messages

• all → forwards entire channel history

7. Set Delay - Speed between messages

text

• 0.5s → Fast (for small jobs)

• 1s → Balanced (recommended)

• 3s → Safe (for large jobs)

8. Choose Filter - What type of messages to forward

 text

• All Messages → Everything

• Media Only → Photos, videos, documents

• Text Only → Only text messages

• Photos Only → Only photos

• Videos Only → Only videos

• Docs Only → Only files

Click "▶️ Start Now"

# 📱 Features
  For Users

📥 Source with Message ID	Start from specific message (e.g., /200)

📤 Multiple Destinations	Forward to multiple channels at once

📩 Message Count	Choose how many messages to forward

⏱️ Delay Control	Set speed between messages

🎛️ Message Filters	All, Media Only, Text Only, Photos, Videos, Docs

🔄 Live Progress	See real-time progress with speed

⏸️ Pause/Resume	Pause and resume anytime

🛑 Cancel Job	Stop running jobs

📋 My Jobs	View and manage all jobs

📜 Job History	See completed jobs

# Do's

Use private channels for personal archives

Add bot as admin in destination channels

Use filters to save space (Media Only for photos/videos)

Check live progress to monitor jobs
📊 Statistics	Track total forwarded messages
