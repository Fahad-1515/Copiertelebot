Frequently asked

How does message count work?

It forwards from the starting message ID forward. e.g. start #50, count=4 → #50, #51, #52, #53.

What does "all" do?

Forwards from your starting message ID to the very end of the channel.

Can I forward to multiple destinations?

Yes! Add multiple destinations in the wizard. Type /done when finished.

What happens if a message fails?

The bot retries automatically (up to 3 times).

Can I pause and resume?

Yes! Use the pause/resume buttons on the status message.

Is my content private?

Yes. Only you and people you add to destinations can see forwarded content.

Does it work with private channels?

Yes, but the bot must be added as an admin to the private channel.

What message types are supported?

All types: text, photos, videos, documents, audio, voice, stickers, GIFs, polls, contacts, locations, albums.
